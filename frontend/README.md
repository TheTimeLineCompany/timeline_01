# V4 Frontend Skeleton

When promoted to runnable UI, adapt the useful V1/V3 reader structure:

- Section click selects the active section.
- Related Information panel fetches cached V4 results.
- No read-path LLM calls.
- Timeline/agent dashboard remain deferred until later phases.

API contract stubs are owned by the backend during Phase 0-2.
